🎯 БЫСТРЫЙ СТАРТ (XRAY)
─────────────────────────────────────────────────────────────────

1️⃣  Скопируй файл:
   sudo cp ru-blocklist-domain.txt /etc/xray/
   
2️⃣  Обнови config.json:
   {
     "routing": {
       "rules": [{
         "type": "field",
         "domain": ["ext:file:///etc/xray/ru-blocklist-domain.txt"],
         "outboundTag": "block"
       }]
     }
   }

3️⃣  Перезагрузи Xray:
   sudo systemctl restart xray
